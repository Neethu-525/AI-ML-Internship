# python file handling tasks
This task contains basic file handling programs in python
## tasks implemented
1. find the longest word in a file
2. count vowels in a file
3. remove duplicates lines from a file

## concepts used
-file handling (read,write)
-exception handling (try-except)
-string operations

## how to run 
create a file named 'data.txt'
add some text inside it
run the python file:
python file_handling_tasks.py
